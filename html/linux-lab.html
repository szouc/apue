CS631 -- Advanced Programming in the UNIX Environment - Linux Lab

All software development will be done on the linux-lab.cs.stevens-tech.edu systems. It is your responsibility to get an account on these systems prior to the start of the class.

All grading will be done on these systems. While you may choose to develop on your own personal host, you should make sure that your code compiles and runs flawlessly on these systems.

In order to get an account on linux-lab.cs.stevens-tech.edu, you need to:

  • go to http://www.stevens.edu/servicedesk
  • sign in (upper right)
  • select 'Services' from the gray navigation bar
  • select 'Computer Science Department Support' from the categories
  • select 'Computer Science Account Requests'
  • select 'Contact CS Dept' on the right
  • fill in the request

Access to linux-lab.cs.stevens.edu is via ssh(1), using a load balancing mechanism that will drop you into any one of a handful of systems. As noted on the course website, you are expected to be familiar with operating in this environment.
