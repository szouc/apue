Extra Credit: implement revline(1)

Problem assignment

Implement the revline(1) command from the in class final. revline(1) reverses the contents of an arbitrarily large text file line by line (but does not reverse the words in the file).

Example:

$ cat file
this is line 1
this is line 2
this is line 3
this is line 4
$ revline file
this is line 4
this is line 3
this is line 2
this is line 1
$
