CS631 -- Advanced Programming in the UNIX Environment - HW#1

HW#1: code reading

Summary

The objective of this assignment is for you to get used to read (and understand!) code. To this end, please take a look at the NetBSD source code for programs installed under /bin, pick two programs of your liking and inspect the code. Then, read the corresponding code of the same programs in Linux or Open Solaris Illumos. Compare the different implementations. What do they have in common? What makes them different?

Deliverables and Due Date

You will submit a single plain text file created with a unix text editor on a unix system called "HW1-<username>" (where "<username>" is your Stevens username). In that file write down (in proper English, spell-checked and with valid grammar):

  • which editor you used to create the document
  • which programs you compared
  • how/why you chose them
  • how you retrieved the source code to the programs in question
  • what you have found out about the implementations

Note: we are not looking for a line-by-line comparison stating that one file has 30 lines of comments while another has 32. Focus on how the different implementations solved the same problem. If there are notable differences, comment on what the reasons for the differences might be.
