CS810 -- Advanced Programming in the UNIX Environment - Fall 2010 - HW#3

HW#3: fill in SWS framework

Summary

The objective of this assignment is for you to write the main framework of your final project. Since the actual functionality of the final project will be implemented in a shared library, you can already build your main program using the (currently) known API defined in sws.h and link against the stub library provided to you here.

Problem assignment

Download all files in this directory. Edit the file main.c to implement all noted functionality. That is, parse all given command-line options, dæmonize if appropriate (ie not in debugging mode), call sws_init, and enter the main loop:

  • initialize a socket to listen on the desired network address(es) and port
  • listen on the socket
  • forever loop, accepting or selecting connections
  • fork a new process for each new connection
  • call sws_request, passing it the socket to process the request
  • close the socket and exit the child

Deliverables and Due Date

You will submit the single file main.c.
